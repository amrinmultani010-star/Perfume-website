# Perfume Store Website

## Overview

A simple **e-commerce perfume store** made using HTML, CSS, and JavaScript. Users can view products, add them to cart, change quantity, remove items, and proceed to checkout.

## Features

* Product listing
* Add to cart
* Increase / decrease quantity
* Remove items from cart
* Cart total calculation
* Payment options: Paytm, Google Pay, PhonePe, Cash on Delivery

## Technologies

* HTML
* CSS
* JavaScript
* LocalStorage

## How to Run

1. Download the project.
2. Open the folder.
3. Run **index.html** in your browser.
